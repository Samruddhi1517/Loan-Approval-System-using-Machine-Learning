## Project Description

Loan Approval Prediction System is a machine learning based web application developed using the Random Forest algorithm. The model was trained in Jupyter Notebook using historical loan applicant data and integrated into a Flask web application.

The system predicts whether a loan application will be **Approved** or **Rejected** based on applicant details such as age, income, credit score, employment status, marital status, number of dependents, and existing loans.
